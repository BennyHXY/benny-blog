    一般公式

    1. ddx(c)=0\frac{d}{dx} (c) = 0dxd​(c)=0

    2. ddx[cf(x)]=cf′(x)\frac{d}{dx}[c f(x)] = c{f}'(x)dxd​[cf(x)]=cf′(x)

    3. ddx[f(x)+g(x)]=f′(x)+g′(x)\frac{d}{dx}[f(x) + g(x)] = {f}'(x) + {g}'(x)dxd​[f(x)+g(x)]=f′(x)+g′(x)

    4. ddx[f(x)−g(x)]=f′(x)−g′(x)\frac{d}{dx}[f(x) - g(x)] = {f}'(x) - {g}'(x)dxd​[f(x)−g(x)]=f′(x)−g′(x)

    5. ddx[f(x)g(x)]=f(x)g′(x)+g(x)f′(x)（函数积的求导法则）\frac{d}{dx}[f(x) g(x)] = f(x){g}'(x) + g(x){f}'(x) \text{（函数积的求导法则）}dxd​[f(x)g(x)]=f(x)g′(x)+g(x)f′(x)（函数积的求导法则）

    6. ddx[f(x)g(x)]=g(x)f′(x)−f(x)g′(x)[g(x)]2（函数商的求导法则）\frac{d}{dx}[\frac{f(x)}{g(x)}] = \frac{g(x){f}'(x)-f(x){g}'(x)}{[g(x)]^{2} } \text{（函数商的求导法则）}dxd​[g(x)f(x)​]=[g(x)]2g(x)f′(x)−f(x)g′(x)​（函数商的求导法则）

    7. ddxf(g(x))=f′(g(x))g′(x)（链式法则）\frac{d}{dx}f(g(x)) = {f}'(g(x)){g}'(x) \text{（链式法则）}dxd​f(g(x))=f′(g(x))g′(x)（链式法则）

    8. ddx(xn)=nxn−1（幂函数的求导法则）\frac{d}{dx}(x^{n}) = n {x}^{n - 1} \text{（幂函数的求导法则）}dxd​(xn)=nxn−1（幂函数的求导法则）

6. $\frac{d}{dx}[\frac{f(x)}{g(x)}] = \frac{g(x){f}'(x)-f(x){g}'(x)}{[g(x)]^{2} }  \text{（函数商的求导法则）}$

7. $\frac{d}{dx}f(g(x)) =  {f}'(g(x)){g}'(x) \text{（链式法则）}$

8. $\frac{d}{dx}(x^{n}) = n {x}^{n - 1}  \text{（幂函数的求导法则）}$
