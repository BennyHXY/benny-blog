### 统计色号

```python
from PIL import Image

import matplotlib.pyplot as plt
import numpy as np

plt.rcParams['font.sans-serif'] = ['SimHei']  # 设置中文字体为黑体
plt.rcParams['axes.unicode_minus'] = False    # 解决负号显示问题

star = Image.open("./star.png")
nd_star = np.array(star)

colors, counts = np.unique(nd_star, return_counts=True)
print(colors, counts)

l = len(colors)

for i in range(l):
    if(counts[i] >= 20):
        plt.text(1,28,
                 "在30×30 (共900个像素点) 的星星图片中，\n 色号{0}出现了{1}次".format(colors[i], counts[i]),
                 bbox=dict(boxstyle="round",
                   ec=(1., 0.5, 0.5),
                   fc=(1., 0.8, 0.8),
                   ),
                 size=18,
                 )
        plt.spy(nd_star == colors[i])
        plt.show()


```

## 小结

通过本次练习，我稍微熟悉了matplotlib和numpy的一些使用。并且在AI的引导下，我了解了一些信息：1. 在 NumPy 和 Matplotlib 的图像表示约定里，数组维度顺序永远是 （高, 宽, 通道） 或 （高, 宽）; 2. `jpg` 对应的数组形状为 (高, 宽, 3) , 其中3对应rgb的三个维度， `png` / `gif` 对应的数组形状为(高, 宽), 因为它们的总颜色数较少，直接映射到色号是一种更好的存储方式，而不是rgb的三个维度。通过不断实践和学习，我们可以进一步掌握图像处理的原理和机制，提高自己的基本素养，为后续的学习奠定基础。
