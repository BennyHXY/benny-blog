# 数学 - 积分表

## 基本公式

:::preview 展开查看latex代码

1. $\int u\mathrm{d}v = uv - \int v \mathrm{d}u$

2. $\int u^n\mathrm{d}u = \frac{u^{n+1}}{n+1} + C, n \ne -1$

3. $\int \frac{\mathrm{d}u}{u} = \ln |u| + C$

4. $\int e^u \mathrm{d}u = e^u + C$

5. $\int b^u \mathrm{d}u = \frac{b^u}{\ln b}  + C$

6. $\int \sin u \mathrm{d}u = - \cos u + C$

7. $\int \cos u \mathrm{d}u = \sin u + C$

8. $\int \sec^{2}u \mathrm{d}u  = \tan u + C$

9. $\int \csc^2 u \mathrm{d}u = -\cot u + C$

10. $\int \sec u \tan u \mathrm{d}u = \sec u + C$

11. $\int \csc u \cot u \mathrm{d}u = - \csc u + C$

12. $\int \tan u \mathrm{d}u = \ln |\sec u| + C$

13. $\int \cot u \mathrm{d}u = \ln |\sin u| + C$

14. $\int \sec u \mathrm{d}u = \ln |\sec u + \tan u| + C$

15. $\int \csc u \mathrm{d}u = \ln |\csc u + \cot u| + C$

16. $\int \frac{\mathrm{d}u }{\sqrt{a^2-u^2} } = \sin^{-1}\frac{u}{a} + C, a > 0$

17. $\int \frac{\mathrm{d}u }{a^2+u^2} = \frac{1}{a}\tan^{-1}\frac{u}{a} + C$

18. $\int \frac{\mathrm{d}u }{u \sqrt{u^2-a^2}} = \frac{1}{a}\sec^{-1}\frac{u}{a} + C$

19. $\int \frac{\mathrm{d}u }{a^2 - u^2} = \frac{1}{2a} \ln |\frac{u + a}{u - a} | + C$

20. $\int \frac{\mathrm{d}u }{u^2 - a^2} = \frac{1}{2a} \ln |\frac{u - a}{u + a}| + C$

:::
