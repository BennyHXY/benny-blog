# 数学 - 导数公式

### 一般公式

:::preview 展开查看latex代码

1. $\frac{d}{dx} (c) = 0$

2. $\frac{d}{dx}[c f(x)] = c{f}'(x)$

3. $\frac{d}{dx}[f(x) + g(x)] = {f}'(x) + {g}'(x)$

4. $\frac{d}{dx}[f(x) - g(x)] = {f}'(x) - {g}'(x)$

5. $\frac{d}{dx}[f(x) g(x)] = f(x){g}'(x) + g(x){f}'(x) \text{（函数积的求导法则）}$

6. $\frac{d}{dx}[\frac{f(x)}{g(x)}] = \frac{g(x){f}'(x)-f(x){g}'(x)}{[g(x)]^{2} }  \text{（函数商的求导法则）}$

7. $\frac{d}{dx}f(g(x)) =  {f}'(g(x)){g}'(x) \text{（链式法则）}$

8. $\frac{d}{dx}(x^{n}) = n {x}^{n - 1}  \text{（幂函数的求导法则）}$

:::

### 指数函数和对数函数

:::preview 展开查看latex代码

9. $\frac{d}{dx}(e^{x}) = e^{x}$

10. $\frac{d}{dx}(b^{x}) = b^{x}\ln b$

11. $\frac{d}{dx} \ln \left | x \right | = \frac{1}{x}$

12. $\frac{d}{dx} (\log_{b}{x} ) = \frac{1}{x \ln b}$

:::

### 三角函数

:::preview 展开查看latex代码

13. $\frac{d}{dx} (\sin x ) = \cos x$

14. $\frac{d}{dx} (\cos x ) = - \sin x$

15. $\frac{d}{dx} (\tan x ) = {\sec}^{2} x$

16. $\frac{d}{dx} (\csc x ) = - \csc x \cot x$

17. $\frac{d}{dx} (\sec x ) = - \sec x \tan x$

18. $\frac{d}{dx} (\cot x ) = - {\csc}^{2} x$

:::

### 反三角函数

:::preview 展开查看latex代码

19. $\frac{d}{dx} ({\sin}^{-1} x) = \frac{1}{\sqrt{1 - x^2} }$

20. $\frac{d}{dx} ({\cos}^{-1} x) = -\frac{1}{\sqrt{1 - x^2} }$

21. $\frac{d}{dx} ({\tan}^{-1} x) = \frac{1}{1 + x^2 }$

22. $\frac{d}{dx} ({\csc}^{-1} x) = -\frac{1}{x \sqrt{x^2 - 1} }$

23. $\frac{d}{dx} ({\sec}^{-1} x) = \frac{1}{x \sqrt{x^2 - 1} }$

24. $\frac{d}{dx} ({\cot}^{-1} x) = -\frac{1}{1 + x^2 }$

:::
